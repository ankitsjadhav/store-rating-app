## üíª Local Setup

1.  **Clone & Install:**

    ```bash
    git clone https://github.com/YOUR_USERNAME/YOUR_REPO.git
    cd server && npm install
    cd ../client && npm install
    ```

2.  **Environment Variables (`.env`):**

    - **Server:** `DATABASE_URL`, `JWT_SECRET`
    - **Client:** `VITE_API_URL=http://localhost:5000/api`

3.  **Run:**
    - Server: `npm start`
    - Client: `npm run dev`
